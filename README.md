# House Price Prediction using Linear Regression

## Objective
Predict house prices based on:
- Square Footage
- Number of Bedrooms
- Number of Bathrooms

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Model Performance
- R² Score: 0.6341
- MAE: 35788.06

## Dataset
Kaggle House Prices Dataset
